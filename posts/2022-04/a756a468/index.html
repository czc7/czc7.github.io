The Modified Embedded Atom Method was formulated by Baskes (M.I.Baskes, Phys.Rev.B 46 (1992) 2727, M.I.Baskes, Mater.Chem.Phys. 50 (1997) 152). It consists of a generalization of the EAM potentials by including angular terms [1].

嵌入能部分的公式

总能量.

E=i∑​Ei​(1-1)

原子 i 的能量.

Ei​=Fi​(ρˉ​i​)+21​j=i∑​ϕij​(rij​)(1-2)

原子 i 的嵌入能.

Fi​(ρˉ​i​)={Ai​Ei0​ρˉ​i​ln(ρˉ​i​)−Ai​Ei0​ρˉ​i​​ if ρˉ​i​⩾0 if ρˉ​i​<0​(1-3)

背景电子密度 ρˉ​i​ 及其分量 ρi(m)​,m=0,1,2,3. ti(m)​,m=1,2,3 是权重因子.

ρˉ​i​=ρi0​ρi(0)​​G(Γi​)(1-4)

Γi​=m=1∑3​ti(m)​(ρi(0)​ρi(m)​​)2(1-4.b)

公式 (1-4) 中背景电子密度参考值 ρi0​ 不是权重因子的 ti(m)​ 的函数.

Newer versions of lammps, by default, use the single-element values of the t parameters to compute the background reference density. This is the proper way to compute these parameters. Earlier versions of lammps used an alloy mixture averaged value of t to compute the background reference density. Setting mixture_ref_t=1 gives the old behavior. WARNING: using mixture_ref_t=1 will give results that are demonstrably incorrect for second-neighbor MEAM, and non-standard for first-neighbor MEAM[2].

G 可以选择不同的形式，lammps 提供的例子使用的是:

G(Γi​)=exp(2Γi​​)(1-4.c)

ρi(m)​,m=0,1,2,3 是背景电子密度的分量，原子 j 是原子 i 的近邻，rijα​ 表示 rij​ 空间上的某个分量，rijα​,rijβ​ 可以表示相同分量，也可以不同.

ρi(0)​=σi0​(1-5.a)

(ρi(1)​)2=α∑​(σi1​(α))2(1-5.b)

(ρi(2)​)2=α,β∑​(σi2​(α,β))2−31​(σi2b​)2(1-5.c)

(ρi(3)​)2=α,β,γ∑​(σi3​(α,β,γ))2−53​α∑​(σi3b​(α))2(1-5.d)

σim[b]​,m=0,1,2,3 是原子 i 所有近邻 j 在 i 位置的电子密度乘以角动量（angular）权重的求和.

σi0​=j=i∑​ρja(0)​(rij​)(1-6.a)

σi1​(α)=j=i∑​ρja(1)​rij​rijα​​(1-6.b)

σi2​(α,β)=j=i∑​ρja(2)​rij2​rijα​rijβ​​(1-6.c)

σi2b​=j=i∑​ρja(2)​(rij​)(1-6.c.b)

σi3​(α,β,γ)=j=i∑​ρja(3)​rij3​rijα​rijβ​rijγ​​(1-6.d)

σi3b​(α)=j=i∑​ρja(3)​rij​rijα​​(1-6.d.b)

ρja(m)​,m=0,1,2,3 为原子 j 在原子 i 位置处的电子密度.

ρja(m)​(rij​)=ρj0​Sij​exp[−βj(m)​(rj0​rij​​−1)](1-7)

公式 (1-7) 中，Sij​(rij​,rik​,rjk​) 除了原子 i 的近邻 j 相关，也与原子 i 的除 j 以外的所有近邻 k 相关. 下面不考虑 Sij​ 与 k 相关.

公式 (1-4.b) 中使用的权重因子 ti(m)​ 的定义如下. t0,j(m)​ 是与原子 j 的类型相关的常数.

ti(m)​=ti,b(m)​ti,a(m)​​(1-8)

分子:

ti,a(m)​=j=i∑​t0,j(m)​ρja(0)​(1-8.a)

分母:

ti,b(m)​=j=i∑​(t0,j(m)​)2ρja(0)​(1-8.b)

公式 (1-1) ~ (1-8) 是原子的坐标到嵌入能的全部计算.

偏微分

不考虑 Sij​ 与 k 相关，嵌入能部分对力的计算如下.

∂rijδ​∂Fi​​​=∂ρˉ​i​∂Fi​​[∂ρi(0)​∂ρˉ​i​​∂rij​∂ρi(0)​​∂rijδ​∂rij​​+m=1∑3​∂(ρi(m)​)2∂ρˉ​i​​(∂rij​∂(ρi(m)​)2​rijδ​∂rij​​+∂rijδ​∂(ρi(m)​)2​)+∂ti(m)​∂ρˉ​i​​∂rij​∂ti(m)​​∂rijδ​∂rij​​]=∂ρˉ​i​∂Fi​​[∂ρi(0)​∂ρˉ​i​​∂rij​∂ρi(0)​​+m=1∑3​∂(ρi(m)​)2∂ρˉ​i​​∂rij​∂(ρi(m)​)2​+∂ti(m)​∂ρˉ​i​​∂rij​∂ti(m)​​]∂rijδ​∂rij​​+∂ρˉ​i​∂Fi​​m=1∑3​∂(ρi(m)​)2∂ρˉ​i​​∂rijδ​∂(ρi(m)​)2​​(2-1)

通过公式 (1-3) 计算嵌入能对背景电子密度的偏微分.

∂ρˉ​i​∂Fi​​={Ai​Ei0​(1+ρˉ​i​)−Ai​Ei0​​ if ρˉ​i​⩾0 if ρˉ​i​<0​(2-2)

通过公式 (1-4) 计算背景电子密度对其分量和权重因子的偏微分.

公式 (1-4.c) 的偏微分：

∂Γi​∂G​=2G(Γi​)​(2-3)

首先是对 ρi(0)​ 的偏微分.

∂ρi(0)​∂ρˉ​i​​=ρi0​1​(G(Γi​)+ρi(0)​∂Γi​∂G​∂ρi(0)​∂Γi​​)

∂ρi(0)​∂Γi​​=−2ρi(0)​Γi​​

∂ρi(0)​∂ρˉ​i​​=ρi0​1​(G(Γi​)−2∂Γi​∂G​Γi​)=ρi0​G(Γi​)(1−Γi​)​(2-4)

其次是对 (ρi(m)​)2 的偏微分.

∂(ρi(m)​)2∂Γi​​=(ρi(0)​)2ti(m)​​

∂(ρi(m)​)2∂ρˉ​i​​=ρi0​ρi(0)​​∂Γi​∂G​∂(ρi(m)​)2∂Γi​​=ρi0​ρi(0)​ti(m)​​∂Γi​∂G​(2-5)

最后是对权重因子 ti(m)​ 的偏微分.

∂ti(m)​∂Γi​​=(ρi(0)​)2(ρi(m)​)2​

∂ti(m)​∂ρˉ​i​​=ρi0​ρi(0)​​∂Γi​∂G​∂ti(m)​∂Γi​​=ρi0​ρi(0)​(ρi(m)​)2​∂Γi​∂G​(2-6)

计算公式 (1-5) 的偏微分如下：

∂σi0​∂ρi(0)​​=1(2-7.a)

∂σi1​(α)∂(ρi(1)​)2​=α∑​2σi1​(2-7.b)

∂σi2​(α,β)∂(ρi(2)​)2​=α,β∑​2σi2​(2-7.c)

∂σi2b​∂(ρi(2)​)2​=−31​2σi2b​(2-7.c.d)

∂σi3​(α,β,γ)∂(ρi(2)​)2​=α,β,γ∑​2σi3​(2-7.d)

∂σi3b​∂(ρi(2)​)2​=−53​α∑​2σi3b​(2-7.d.b)

计算公式 (1-6) 对 ρja(m)​ 的偏微分, 结果即角动量张量.

∂ρja(m)​∂σim[b]​​=1,rij​rijα​​,(rij​)2rijα​rijβ​​,1,(rij​)3rijα​rijβ​rijγ​​,rij​rijα​​(2-8)

将 ρja(m)​ 在公式 (1-6) 中展开，计算 (1-6) 对 rij​ 的偏微分.

∂rij​∂σim[b]​​​=∂rij​∂ρja(0)​​,rijα​rij2​rij​∂rij​∂ρja(1)​​−ρja(1)​​,rijα​rijβ​rij3​rij​∂rij​∂ρja(2)​​−2ρja(2)​​,∂rij​∂ρja(2)​​,rijα​rijβ​rijγ​rij4​rij​∂rij​∂ρja(3)​​−3ρja(3)​​,rijα​rij2​rij​∂rij​∂ρja(3)​​−ρja(3)​​​(2-9)

计算公式 (1-6) 对 rijδ​ 的偏导.

∂rijδ​∂σim[b]​​​=0,rij​ρja(1)​​,(rij​)2ρja(2)​​rijβ​,2(rij​)2ρja(2)​​rijδ​,0,(rij​)3ρja(3)​​rijβ​rijγ​,2(rij​)3ρja(3)​​rijδ​rijγ​,3(rij​)3ρja(3)​​rijδ2​,rij​ρja(3)​​​(2-10)

计算公式 (1-7) 的偏微分.

∂rij​∂ρja(m)​​=−rj0​ρj0​βj(m)​​Sij​exp[−βj(m)​(rj0​rij​​−1)]=−rj0​βj(m)​​ρja(m)​(2-11)

通过公式 (1-8) 计算 ti(m)​ 对 ρja(0)​ 的偏微分.

∂ρja(0)​∂ti(m)​​=(ti,b(m)​)2ti,b(m)​t0,j(m)​−ti,a(m)​(t0,j(m)​)2​=ti,b(m)​t0,j(m)​−ti(m)​(t0,j(m)​)2​(2-12)

公式 (2-2) ~ (2-12) 代入 (2-1) 中即可.

张量形式

注: 使用符号 ⊙ 表示张量 element wise 的乘法，dimK⋅​ 表示张量消去维度 K 的点积，以上两个操作如果左右表达式维度不同，则复制扩充维度.
